# Documentation

Architecture and deployment documentation for the uKeyTr platform.

## Key Files

| File | Purpose |
|------|---------|
| `ARCHITECTURE.md` | System architecture overview, service topology, data flow |
| `AGENT_ARCHITECTURE.md` | AI agent swarm design — agents, consensus voting, memory system |
| `API_REFERENCE.md` | Full API endpoint reference for backend and AI engine |
| `DEPLOYMENT.md` | Railway deployment guide, environment variables, health checks |
| `ENVIRONMENT_VARIABLES.md` | Complete environment variable reference table |

## Reading Order

1. **`ARCHITECTURE.md`** — Start here for the big picture
2. **`AGENT_ARCHITECTURE.md`** — Deep dive into the AI swarm
3. **`API_REFERENCE.md`** — All endpoints with request/response examples
4. **`DEPLOYMENT.md`** — How to deploy to Railway
5. **`ENVIRONMENT_VARIABLES.md`** — Quick reference for all config

# Domain Models

Domain entities and value objects.

## Key Files

| File | Purpose |
|------|---------|
| `AgentModels.cs` | AI swarm models — AgentHeartbeat, SwarmStatus, SwarmSignal, ConsensusVote, PortfolioPosition |
| `BacktestModels.cs` | Backtest entities — BacktestRun, BacktestResult, TradingSignal |
| `IndicatorModels.cs` | Indicator computation results |
| `MarketDataModels.cs` | OHLCV candle data, ticker info |
| `UserModels.cs` | User account and subscription models |

# App Routes

Next.js 16 App Router pages. Each folder is a route segment.

## Key Pages

| Route | Folder | Description |
|-------|--------|-------------|
| `/` | `page.tsx` | Landing page |
| `/auth` | `auth/` | Login and registration |
| `/swarm` | `swarm/` | AI Agent Swarm Dashboard |
| `/signals` | `signals/` | Trading signal history |
| `/portfolio` | `portfolio/` | Portfolio performance |
| `/performance` | `performance/` | 90-day analysis |
| `/backtest` | `backtest/` | Strategy backtesting |
| `/indicators` | `indicators/` | Technical indicators view |
| `/data` | `data/` | Market data explorer |
| `/education` | `education/` | Trading education hub |
| `/pricing` | `pricing/` | Subscription plans |
| `/health` | `health/` | Frontend health check |

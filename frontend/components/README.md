# Components

Reusable React components for the trading dashboard.

## AI Swarm Components

| File | Purpose |
|------|---------|
| `AgentStatusCard.tsx` | Agent card with icon, status dot, last heartbeat time |
| `SignalFeed.tsx` | Signal list — direction badges, entry/SL/TP, confidence, status |
| `RiskGauge.tsx` | Visual risk gauge bar with kill switch indicator |
| `PortfolioSummary.tsx` | Stats grid — portfolio value, PnL, win rate, signals |
| `SwarmTopology.tsx` | SVG topology diagram of agent hierarchy |
| `EquityCurveLarge.tsx` | Large SVG equity curve with area fill |

## Trading Components

| File | Purpose |
|------|---------|
| `Navbar.tsx` | Main navigation with desktop and mobile menu |
| `TradingChart.tsx` | Candlestick chart with TradingView-style rendering |
| `IndicatorPanel.tsx` | Indicator selector and overlay panel |
| `TickerBar.tsx` | Live ticker strip |
| `BacktestPanel.tsx` | Backtest configuration and results |
| `PriceAlertForm.tsx` | Price alert creation form |

## Layout Components

| File | Purpose |
|------|---------|
| `Hero.tsx` | Landing page hero section |
| `Features.tsx` | Feature showcase grid |
| `Footer.tsx` | Site footer |
| `PricingCards.tsx` | Subscription tier comparison |
